# 🎮 Tic-Tac-Toe Game

A simple Tic-Tac-Toe game built using **HTML, CSS, and JavaScript**.

## 📌 Features

- Two-player game (X vs O)
- Displays the current player's turn
- Detects the winner
- Detects a draw
- Highlights the winning boxes
- Restart button to play again

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript

## 📂 Project Structure

```
Tic-Tac-Toe/
│── index.html
│── style.css
│── script.js
```

## ▶️ How to Run

1. Download or clone the project.
2. Open the project folder.
3. Double-click `index.html` or open it in your web browser.
4. Start playing!

## 🎮 How to Play

1. Player **X** starts the game.
2. Players take turns placing **X** and **O**.
3. The first player to match three symbols in a row, column, or diagonal wins.
4. If all boxes are filled and no player wins, the game ends in a draw.
5. Click the **Restart** button to play again.
